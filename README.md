# 🚀 CV Builder

Modern CV builder with real-time preview, customizable sections, and PDF export.

Built with **Next.js, TypeScript, Tailwind CSS, and PostgreSQL (Neon)**.

---

## ✨ Features

- 📝 Real-time CV editing
- 👀 Live preview with pagination
- 📄 Export to PDF
- ⚡ Smooth UI with animations (Framer Motion)
- 🧩 Modular and scalable architecture
- 🌙 Clean dark mode UI

---

## 🛠 Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS, ShadCN UI
- **State Management:** useReducer
- **Backend:** Next.js Server Actions
- **Database:** PostgreSQL (Neon)
- **Deployment:** Vercel

---

## 📦 Getting Started

```bash
git clone https://github.com/learnbit/cv-builder.git
cd cv-builder
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a .env.local file:

```bash
DATABASE_URL=your_database_url_here
```

---

## 🚀 Deployment

This project is optimized for deployment on Vercel.

1. Push to GitHub
2. Import project in Vercel
3. Add environment variables
4. Deploy
